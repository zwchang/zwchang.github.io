Analysis and Prediction of Apartment Sales in Lund, Sweden

The aim of this project is to provide a thorough analysis and eventually build model using machine learning algorithms to predict the housing prices in the city of Lund where I am currently living. I hope this project could not only provide a general overview of the recent Lund housing market, but also answer business-like questions for the potential housing buyers/sellers like:

  1. When is the best time to sell/buy an apartment/house?

  2. If you are going to sell a property, which agency (or even more progressively, whom) you should call?

  3. What is the reasonable price for a given property with parameters like location, living area size, number of rooms, floor number, year of build, etc.?

Assignments from Andrew Ng's machine learning online course rewritten using Python

Here I gradually upload Python solutions for the exercises of the famous online machine learning course from Andrew Ng. Like many other, this course introduced me into the ML and AI world. For the educational purpose, here I try not only rewrite all original Matlab codes in Python but also implement corresponding algorithms in Scikit-learn as a comparison. AlgoExpert Solutions

Both software engineer and data scientist profession require solid foundation of algorithms and data structure knowledge and programming skills. The most efficient way to improve is through extensive practice, either working on real projects or solving coding interview questions. A lot of programmers practice this on Leetcode. I chose a slightly different platform AlgoExpert where they select about 160 common coding interview questions. For each of them, they also provide video explanation about the solution. So far, I've solved 20 questions and I will keep going. In the meantime, I will also gradually upload the questions together with my solutions to the github repo.
